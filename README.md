# Research Methodology Toolkit

Interactive research methodology and statistical analysis tools for emerging business and social science researchers.

## Available Tools

- **Research Topic Builder** (`topic-builder.html`): Step-by-step guide for constructing research problem statements for BBA programs

## Planned Tools

- Hypothesis Generator
- Literature Review Matrix
- Sampling Calculator
- Statistical Analysis Tools

## File Naming Structure

To maintain consistency as the toolkit grows, all tools follow kebab-case naming conventions:

### Research Methodology Tools

- `topic-builder.html` - Research topic construction (current)
- `hypothesis-generator.html` - Formulating research hypotheses
- `literature-matrix.html` - Organizing literature reviews
- `sampling-calculator.html` - Sample size calculations

### Statistical Analysis Tools

- `descriptive-stats.html` - Basic descriptive statistics
- `correlation-analyzer.html` - Correlation analysis
- `regression-tool.html` - Regression analysis helper
- `anova-calculator.html` - ANOVA computations
- `reliability-tester.html` - Cronbach's alpha calculator

### Data Collection Tools

- `questionnaire-builder.html` - Survey design and construction
- `scale-validator.html` - Measurement scale validation
